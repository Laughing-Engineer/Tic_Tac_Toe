# Tic-Tac-Toe AI

## 📌 Project Description

This project is an AI-powered Tic-Tac-Toe game developed using Python. The game allows a human player to compete against an AI that uses the **Minimax Algorithm** to make the best possible move. The AI is designed to be unbeatable, ensuring it either wins or forces a draw.

## 🎯 Features

* Human vs AI gameplay
* AI uses the Minimax Algorithm
* Unbeatable AI
* Detects win, lose, and draw conditions
* Simple command-line interface

## 🛠 Technologies Used

* Python 3
* Minimax Algorithm

## 💡 Algorithm Used

The AI uses the **Minimax Algorithm**, a decision-making algorithm from game theory that evaluates all possible moves and selects the optimal one. This makes the AI unbeatable.
